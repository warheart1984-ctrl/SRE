# Mythar IP Protection

**Owner:** Jon Halstead  
**Asset:** Mythar language, lexicon, cosmology, and branded artifacts

Mythar is a constitutionally governed proto-language created by Jon Halstead.  
This directory documents the intellectual property protections that ensure Mythar remains proprietary while the CIH/SRE/LRL ecosystem remains open for academic and scientific use.

---

## 1. Constitutional protection

Mythar lexicon entries, grammar structures, writing system glyphs, and cosmological mappings are protected through:

- **CIH Constitutional Kernel** — defines governance rules and invariants (`docs/architecture/CIH_ConstitutionalKernel_v1.md`)
- **CEL Evidence Plane** — provides hash-chained, append-only lineage (`docs/governance/CEL_Charter_v01.md`)
- **Sovereign Certificates** — cryptographically attest reconstruction authenticity (`certificates/Mythar_Certificate_Template.md`)
- **FRA Pipeline** — produces reproducible reconstruction lineage (`docs/architecture/FRA_Methodology.md`)

Architecture diagram: `docs/architecture/assets/CGLR_Architecture_v1.png` · `docs/architecture/CGLR_Architecture_v1.md`

These systems create **cryptographic provenance**, preventing unauthorized duplication.

## 3. Academic protection

CIH, LRL, CRA, CEL, and FRA are open specifications.  
Mythar is **not** part of CIH and remains proprietary.

Academic institutions may study:

- CIH governance
- CEL evidence lineage
- FRA reconstruction methodology
- LRL domain architecture

But Mythar itself is licensed, not open.

See: `docs/submissions/Cambridge_SubmissionPackage_v1.md`, `docs/governance/CIH_Mythar_RelationshipCharter_v1.md`

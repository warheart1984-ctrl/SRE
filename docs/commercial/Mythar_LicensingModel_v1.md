# Mythar Licensing Model v1.0

**Owner:** Jon Halstead  
**Product:** Mythar (proprietary CIH-governed proto-language)

---

## 1. License Types

### 1.1 Personal License

- Non-commercial use
- Includes grammar + lexicon (as licensed)
- No redistribution

### 1.2 Creator License

- Authors, world-builders, indie developers
- Use Mythar in books, games, worlds
- Attribution required
- No sublicensing

### 1.3 Studio License

- Film, AAA games, VR studios
- Full commercial rights within licensed scope
- Sublicensing allowed per agreement
- Includes custom lexicon generation (as contracted)

### 1.4 Enterprise License

- AI companies, platform licensors
- Language model embedding rights
- Custom Mythar corpora
- Annual renewal

---

## 2. License Rights

Licensees MAY (per tier):

- Use Mythar lexicon
- Use Mythar grammar
- Use Mythar writing system (when published)
- Create derivative works (with attribution where required)
- Commercial distribution (Studio / Enterprise only)

---

## 3. Restrictions

Licensees MUST NOT:

- Claim Mythar as an original unlicensed creation
- Remove constitutional lineage metadata from governed artifacts
- Alter or falsify CEL evidence records
- Sell, sublicense, or redistribute Mythar independently (except as permitted by Studio/Enterprise terms)

---

## 4. Enforcement

| Mechanism | Role |
|-----------|------|
| **CEL lineage** | Proves authorship and change history |
| **CIH certification** | Proves governed reconstruction authenticity |
| **Dantomax / attestation** | Cryptographic cluster and evidence anchors |
| **Legal** | Copyright, trademark, license agreements, DMCA |

Detail: `docs/commercial/Mythar_IPProtectionStrategy_v1.md`

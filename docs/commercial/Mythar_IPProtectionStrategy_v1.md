# Mythar IP Protection Strategy v1.0

**Owner:** Jon Halstead  
**Asset:** Mythar language, lexicon, cosmology, and branded artifacts

---

## 1. Constitutional protection

Mythar lexicon entries are:

- CEL-anchored (`docs/governance/CEL_Charter_v01.md`)
- Hash-chained
- Constitutionally certified where applicable
- Evidence-bound (FAC-E / FAC-V)

This provides **cryptographic provenance** complementing traditional IP law.

---

## 2. Legal protection

| Instrument | Target |
|------------|--------|
| **Copyright** | Mythar lexicon, grammar, published materials |
| **Trademark** | Mythar name |
| **Trademark** | Mythar writing system (when published) |
| **License agreements** | Per `docs/commercial/Mythar_LicensingModel_v1.md` |
| **DMCA** | Unauthorized distribution takedowns |

## 3. Technical protection

| Control | Implementation |
|---------|----------------|
| CEL lineage | Every governed cluster / certification path |
| Dantomax signatures | Attestation receipts on accepted evidence |
| Immutable reconstruction history | Append-only CEL + governance traces |

Repository lexicon data (`data/mythar_lexicon_v01.json`) is **development reference**; commercial distribution is under license, not open repo terms.

## 4. Academic vs proprietary boundary

| Open (shareable with institutions) | Closed (Mythar IP) |
|-----------------------------------|---------------------|
| CIH Constitutional Kernel | Mythar lexicon clusters |
| LRL Specification | Mythar cosmology / ritual syntax |
| CRA Reference Architecture | Mythar writing system (commercial) |
| FRA methodology | Branded Mythar publications |
| CEL evidence model | Licensed derivative works |

Cambridge and similar institutions receive the **framework**, not Mythar IP.  
See: `docs/submissions/Cambridge_SubmissionPackage_v1.md`

---

## 5. Enforcement workflow

1. Detect unauthorized use (manual, partner report, or automated lineage mismatch).
2. Verify CEL / certification lineage for authentic vs forged artifacts.
3. Issue cease-and-desist or DMCA where applicable.
4. Offer legitimate licensing path via `docs/commercial/Mythar_LicensingModel_v1.md`.

# Mythar Commercialization Plan v1.0

**Status:** Draft  
**Owner:** Jon Halstead  
**Governance:** CIH-governed proprietary language (not part of CIH specification)

---

## Executive Summary

Mythar is the first CIH-governed proto-language with constitutional lineage, CEL-anchored lexicon entries, and a reproducible FRA reconstruction path. It is both a **linguistic asset** and a **commercial product**.

This plan defines Mythar's commercialization strategy across licensing, publishing, services, and ecosystem development.

**Open stack (not Mythar IP):** CIH, LRL, CRA, FRA methodology, CEL evidence model, SRE reference runtime.  
**Proprietary asset:** Mythar lexicon, cosmology, ritual syntax, and branded language artifacts.

See: `docs/governance/CIH_Mythar_RelationshipCharter_v1.md`

---

## 1. Product Definition

Mythar is:

- A governed proto-language
- A synthetic + emergent lexicon
- A constitutional linguistic artifact
- A proprietary IP asset
- A reconstruction domain under CIH

**Mythar is not part of the CIH specification.** It is a commercial language built on CIH.

| Layer | Status |
|-------|--------|
| CIH / LRL / CRA / FRA / CEL | Open constitutional framework |
| SRE | Reference implementation |
| **Mythar** | **Proprietary, licensed** |

Technical reference (non-IP): `docs/architecture/MytharGapFill.md`, `data/mythar_lexicon_v01.json`

## 2. Commercial Pillars

### 2.1 Licensing

Mythar can be licensed to:

- Game studios
- Film/TV studios
- VR world builders
- AI world creators
- Novelists and world-builders
- Linguistics startups
- Spiritual/ritual communities

**Licensing models:**

| Model | Use case |
|-------|----------|
| Per-project license | Single title / world |
| Annual studio license | Ongoing production |
| Enterprise world-building license | Multi-franchise platforms |
| AI-model embedding license | LLM / agent corpora |

### 2.2 Publishing

Mythar can be monetized through:

- Grammar books
- Lexicon volumes
- Writing system guides
- Mythar lore and cosmology
- Sound change charts
- Ritual syntax manuals

Sell or license:

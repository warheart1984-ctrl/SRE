## 1. Phonology

**Consonants:** p, b, t, d, k, g, m, n, r, l, f, s, y, j, q

**Vowels:** a, e, i, o, u (with ritual long forms ā, ē, ī)

**Phonological rules:**

| Change | Context |
|--------|---------|
| ma → mah | Ritual breath extension |
| la → lā | Illumination extension |
| ra → raa | Proclamation intensification |

Sound laws: `docs/mythar/publishable/Mythar_SoundChangeLaws_v1.md`

## 2. Morphology

### 2.1 Roots

Mythar roots are semantic-axis aligned:

| Root | Meaning |
|------|---------|
| ma | mother / breath |
| la | light |
| ra | proclamation |
| ka | power |
| ti | child / emergence |
| fa | blessing |
| yo / ya / yu | divine call |

### 2.2 Compounding

Compounds follow **axis → modifier → intensifier**:

**Example:** `morkfu` = mor (rise) + k (force) + fu (blessing)

## 4. Semantics

Mythar semantics are **axial**, not categorical.

**Axes include:** Light, Heart, Soul, Blessing, Emergence, Threshold, Protection

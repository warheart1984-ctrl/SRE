# MYTHAR PROTO-SCRIPT FONT SPECIFICATION v1.0

**Technical specification for the Mythar writing system font**  
**Owner:** Jon Halstead

---

## 1. Font Overview

The **Mythar Proto-Script Font (MPSF)** is a geometric proto-script typeface expressing Mythar graphemes with constitutional clarity and ceremonial symmetry.

Writing system: `docs/mythar/publishable/Mythar_WritingSystem_v1.md`

---

## 2. Glyph Design Principles

| Principle | Meaning |
|-----------|---------|
| **Axis Linearity** | Vertical strokes = semantic axes |
| **Breath Curvature** | Curved strokes = emergence / blessing |
| **Proclamation Triads** | Triangular motifs = activation |
| **Mother-Breath Loops** | Circular forms = origin |

---

## 3. Glyph Families — Axis Glyphs

| Glyph | Construction |
|-------|--------------|
| MA | central loop + vertical axis |
| LA | vertical axis + radiance marks |
| RA | triangular proclamation mark |
| KA | force-stroke + axis |
| FA | curved blessing stroke |
| TI | upward emergence stroke |
| PA | protective enclosure |
| TOR | boundary gate glyph |

---

## 4. Font Metrics

| Metric | Value |
|--------|-------|
| X-height | 0.72 of full glyph height |
| Ascender | 0.18 |
| Descender | 0.10 |
| Stroke width | 0.06 |
| Ritual long-forms | +0.12 ascender extension |

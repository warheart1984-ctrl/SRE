# MYTHAR WRITING SYSTEM v1.0

**A governed proto-script specification**  
**Owner:** Jon Halstead

---

## 1. Script Overview

The **Mythar Writing System (MWS)** is a governed proto-script whose graphemes align to Mythar’s semantic axes: Light, Heart, Soul, Blessing, Emergence, Threshold, Protection, and Power.

Each grapheme is CEL-anchored, constitutionally certified, and reconstructable through FRA lineage.

## 2. Grapheme Families

### 2.1 Axis Graphemes (Primary)

| Axis | Grapheme | Meaning |
|------|----------|---------|
| Light | LA | illumination, clarity |
| Heart | KA | vital force |
| Soul | RA | proclamation, activation |
| Blessing | FA | giving, yield |
| Emergence | TI | birth, newness |
| Protection | PA | guard, shelter |
| Threshold | TOR | gate, boundary |
| Mother | MA | breath, origin |

Each grapheme has a **long form** for ritual inscriptions (e.g. LĀ, MAH, RAA).

---

## 3. Script Geometry

1. **Axis Linearity** — vertical strokes represent semantic axes  
2. **Breath Curvature** — curved strokes represent emergence or blessing  
3. **Proclamation Triads** — triangular motifs represent activation or divine call  

---

## 4. Writing Direction

| Mode | Direction |
|------|-----------|
| Standard | Left-to-right |
| Ritual | Axis-first (semantic ordering) |
| Ceremonial | Boustrophedon variant for inscriptions |

---

## 5. Grapheme Construction

Each grapheme = **axis stroke** + **modifier stroke** + **intensifier mark** (ritual emphasis)

**Example:** MUFAY = MU (mother-breath) + FA (blessing) + Y (grace mark)

---

## 6. Script Examples

- **LA-TI-MA** — “The light births the mother-breath.”
- **RAA-KA-FA** — “Proclamation empowers blessing.”

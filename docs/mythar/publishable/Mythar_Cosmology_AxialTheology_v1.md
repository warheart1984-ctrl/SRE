## 1. Overview

Mythar cosmology is axial, semantic, and governed. It describes the metaphysical structure underlying Mythar’s lexicon and grammar.

## 2. The Eight Axes

| Axis | Form | Meaning |
|------|------|---------|
| Light | LA | illumination, truth |
| Heart | KA | force, vitality |
| Soul | RA | proclamation, identity |
| Blessing | FA | grace, giving |
| Emergence | TI | birth, creation |
| Mother | MA | breath, origin |
| Protection | PA | shelter, guardianship |
| Threshold | TOR | boundary, transition |

These axes form the **Axial Mandala**.

## 4. Theological Structure

Mythar theology is **non-deific and axial:**

| Axis | Theological role |
|------|------------------|
| Light | revelation |
| Heart | will |
| Soul | proclamation |
| Blessing | grace |
| Emergence | creation |
| Mother | origin |
| Protection | guardianship |
| Threshold | transformation |

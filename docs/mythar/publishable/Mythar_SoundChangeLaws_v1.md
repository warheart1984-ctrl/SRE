## 1. Overview

Mythar sound laws are FRA-derived, CEL-anchored, and constitutionally validated. They describe proto-form evolution across ritual, emergent, and ceremonial registers.

## 2. Core Sound Laws

### 2.1 Breath Extension (MA → MAH)

When a form invokes origin, protection, or ritual breath, **MA** lengthens to **MAH**.

### 2.2 Light Extension (LA → LĀ)

Illumination contexts extend **LA** to **LĀ**.

### 2.3 Proclamation Intensification (RA → RAA)

Proclamation contexts double the vowel.

### 2.4 Divine Glide (YA → YALA / YAMI)

- Divine call + light → **YALA**
- Divine call + smallness → **YAMI**

### 2.5 Emergence Softening (TI → TIN / TINA)

Child/emergence contexts soften **TI**.

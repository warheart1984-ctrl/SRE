# MYTHAR RITUAL SYNTAX v1.0

**The ceremonial grammar of governed language**  
**Owner:** Jon Halstead

---

## 1. Overview

Ritual syntax is the ceremonial grammar of Mythar. It governs invocation, blessing, emergence, and proclamation.

---

## 2. Ritual Ordering

Ritual lines follow:

**Axis → Light → Power → Blessing → Name**

**Example:**  
`ebro la kajya lapio mufay`  
“Emergent bearer of light, divine-affirmed power, seed-light place, mother-breath blessing.”

---

## 3. Ritual Registers

### 3.1 Invocation Register

Call divine or ancestral forces.  
**Structure:** Divine → Light → Heart

### 3.2 Blessing Register

Confer protection or grace.  
**Structure:** Mother → Blessing → Light

### 3.3 Emergence Register

Birth, creation, newness.  
**Structure:** Emergence → Light → Name

---

## 4. Ritual Particles

| Particle | Role |
|----------|------|
| yo | divine call |
| mu | mother-breath |
| ra | proclamation |
| ti | emergence |
| fa | blessing |

---

## 5. Ritual Examples

**Blessing:**  
*Mu fa la ra.* — “Mother blesses light with proclamation.”

**Emergence:**  
*Ti la mufay.* — “Emergence enters blessed light.”
